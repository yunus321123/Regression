# Regression Assignment (Jupyter Notebook)

This repository contains a complete **Regression assignment** implemented using **Python** in **Jupyter Notebook (.ipynb)** format.

The assignment is written in **simple and easy English**, making it suitable for academic submission and easy understanding.

---

## Contents of the Assignment

The notebook covers the following topics:

- Simple Linear Regression  
- Assumptions of Linear Regression  
- Slope and Intercept interpretation  
- Least Squares Method  
- Coefficient of Determination (R² and Adjusted R²)  
- Multiple Linear Regression  
- Multicollinearity and Heteroscedasticity  
- Categorical Variable Encoding  
- Polynomial Regression  
- Model Evaluation and Visualization  

A total of **31 theory questions** are answered in detail.
